# Changelog

All notable changes to UsagePilot will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of UsagePilot
- Active window tracking with Windows API
- Idle detection system
- SQLite local storage
- React frontend with Tailwind CSS
- Dark mode support
- Dashboard with usage charts
- Applications list with detailed breakdown
- Autostart functionality
- Settings page
- Friendly application names display

### Security
- All data stored locally, no external connections
- Privacy-first architecture

## [0.1.0] - 2025-11-20

### Added
- Initial public release
- Core usage tracking functionality
- Beautiful UI with React and Tailwind
- Dark mode support
- Local SQLite database
- Cross-platform support (Windows, macOS, Linux)
